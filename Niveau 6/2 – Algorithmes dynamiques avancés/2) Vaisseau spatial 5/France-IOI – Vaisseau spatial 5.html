Java est un langage de programmation impératif et orienté objet. Même problème que Vaisseau spatial 4, mais il peut maintenant y avoir des malus.

Limites de temps et de mémoire (C++)

  • Temps : 5 s sur une machine à 1 GHz.
  • Mémoire : 2 000 ko.

Contraintes

  • -1000 <= B <= 1000, où B est la valeur d'un bonus.
  • 1 <= N <= 20000, où N est le nombre de lignes du jeu.
  • 1 <= M <= 500, où M est le nombre de colonnes du jeu.
  • 0 <= P < M, où P est la colonne où se trouve votre vaisseau au départ.

Entrée

  • La première ligne de l'entrée contient trois entiers :
    • N, le nombre de lignes.
    • M, le nombre de colonnes.
    • P, l'indice de la colonne de départ.
  • Chacune des N-1 lignes suivantes contient deux entiers, I[i] et V[i], I[i] étant la colonne où se trouve le bonus de la ième ligne, et V[i] indique la valeur de ce bonus. La dernière ligne ne contient toujours pas de bonus.

Sortie

Vous devez écrire une ligne sur la sortie, contenant N-1 entiers, séparés par des espaces. Chaque entier représente un déplacement : -1 pour gauche, 0 pour aller tout droit, 1 pour aller à droite.

Il se peut qu'il y ait plusieurs possibilités; si à un moment on peut faire le meilleur score soit en allant tout droit soit en tournant, on ira tout droit. Si le choix est entre tourner à gauche ou tourner à droite, on ira à gauche.

Exemple

entrée : 

5 3 2
2 10
1 -8
0 6
2 2

sortie : 

0 0 0 0

Poser une question

Si malgré les conseils automatiques vous n'arrivez plus à avancer, n'hésitez pas à poser une question. Les utilisateurs et utilisatrices qui ont déjà résolu ce sujet ou les entraîneurs de France-IOI vous apporteront de l'aide. Ils ne vous donneront pas la solution mais seulement des astuces et des indications pour vous accompagner dans la réalisation de votre programme et vous permettre d'être plus autonome dans la suite de votre progression.

Les personnes qui vous aideront pourront voir toutes vos soumissions de l'onglet Activité ainsi que votre code dans l'éditeur. Vous n'avez donc pas besoin de copier-coller votre code dans votre message, ni les erreurs que vous avez obtenues. Si vous n'avez fait aucune soumission, assurez-vous que vous avez mis dans l'éditeur le programme que vous souhaitez présenter.
