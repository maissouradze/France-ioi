Le problème est encore le même, mais il n'y a plus qu'un seul bonus par ligne, de valeur entre 0 et 1000, et le nombre de lignes et de colonnes est plus grand.

Limites de temps et de mémoire (C++)

  • Temps : 1 s sur une machine à 1 GHz.
  • Mémoire : 2 000 ko.

Contraintes

  • 0 <= B <= 1000, où B est la valeur d'un bonus.
  • 1 <= N <= 20000, où N est le nombre de lignes du jeu.
  • 1 <= M <= 500, où M est le nombre de colonnes du jeu.
  • 0 <= P < M, où P est la colonne où se trouve votre vaisseau au départ.

Entrée

  • La première ligne de l'entrée contient trois entiers :
    • N, le nombre de lignes.
    • M, le nombre de colonnes.
    • P, l'indice de la colonne de départ.
  • Chacune des N-1 lignes suivantes contient deux entiers, I[i] et V[i], I[i] étant la colonne où se trouve le bonus de la ième ligne, et V[i] indique la valeur de ce bonus. La dernière ligne ne contient toujours pas de bonus.

Sortie

Vous devez écrire une ligne sur la sortie, contenant un entier : le score maximal que l'on peut obtenir.

On ne demande plus le chemin correspondant.

Exemple

entrée : 

5 3 2
2 10
1 8
0 6
2 2

sortie : 

24

Il y a trop de lignes pour stocker tout le tableau jeu en mémoire. La solution consiste à partir de la version itérative de Vaisseau spatial 1. Reprenons le coeur de l'algorithme, et adaptons-le aux nouveaux bonus : on met 0 dans toutes les cases de la ligne, sauf à l'endroit du bonus où l'on met la valeur du bonus.

for(int lin = 0; lin < nbLin; lin++)
{ 
   for(int col = 0; col < nbCol; col++)
      jeu[lin][col] = 0;
   jeu[lin][bonus[lin].pos] = bonus[lin].val;
   if (lin > 0)
   {
      for(int col = 0; col < nbCol; col++)
      {
         int meilleur = MOINS_INFINI;
         for(int dep = -1; dep <= 1; dep++)
         {
            int curCol = col + dep;
            if (curCol >= 0 && curCol < nbCol)
               if (jeu[lin-1][curCol] > meilleur)
                  meilleur = jeu[lin-1][curCol];
         }
         jeu[lin][col] += meilleur;
      }
   }
}

Pour remplir une ligne du tableau jeu, on a besoin des valeurs de la ligne en cours et de celles de la ligne juste au-dessus, et c'est tout. On peut donc se permettre "d'oublier" les lignes du dessus. Oublier, celaa veut dire qu'on peut réutiliser l'espace mémoire pour y écrire d'autres données. Au total on a besoin de deux lignes : la ligne que l'on veut construire, et celle située juste au dessus.

On va ainsi pouvoir réduire la taille du tableau meilleurScore pour qu'il n'y ait plus que 2 lignes et non plus MAX_LIN.

int meilleurScore[2][MAX_COL];

Où va-t-on mettre les valeurs de meilleurScore[lin][*] ? Lorsque lin est pair, on va les mettre dans meilleurScore[0][*], et lorsque lin est impair on va mettre dans meilleurScore[1][*]. Lorsqu'on veut calculer les valeurs de la ligne lin, on écrira dans lin modulo 2, que l'on code lin%2. On posera cur égal à lin%2 pour la ligne courante. Tout le code :

#include <stdio.h>
const int MOINS_INFINI = -100000000;
const int MAX_LIN = 20000;
const int MAX_COL = 500;
struct Bonus {
   int pos;
   int val;
};
int nbLin, nbCol, colDepart;
int meilleurScore[2][MAX_COL];
Bonus bonus[MAX_LIN-1];
int main()
{
   scanf("%d %d %d", &nbLin, &nbCol, &colDepart);
   for(int lin = 0; lin < nbLin-1; lin++)
      scanf("%d %d", &bonus[lin].pos, &bonus[lin].val);
   for(int lin = 0; lin < nbLin; lin++)
   { 
      int ind = lin % 2;
      for(int col = 0; col < nbCol; col++)
         meilleurScore[ind][col] = 0;
      if (lin < nbLin-1)
         meilleurScore[ind][bonus[lin].pos] = bonus[lin].val;
      if (lin > 0)
      {
         for(int col = 0; col < nbCol; col++)
         {
            int meilleur = MOINS_INFINI;
            for(int dep = -1; dep <= 1; dep++)
            {
               int curCol = col + dep;
               if (curCol >= 0 && curCol < nbCol)
                  if (meilleurScore[(lin-1)%2][curCol] > meilleur)
                     meilleur = meilleurScore[(lin-1)%2][curCol];
            }
            meilleurScore[ind][col] += meilleur;
         }
      }
   }
   printf("%d\n", meilleurScore[(nbLin-1)%2][colDepart]);
   return 0;
}
